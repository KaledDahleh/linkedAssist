EasyReach Privacy Policy

Last updated: March 24, 2026

Overview

EasyReach is a Chrome extension that helps you draft LinkedIn messages using AI. Your privacy matters — we've designed the extension so that your data stays on your device.

Data we collect

None. EasyReach does not collect, transmit, or store any user data on external servers. We have no backend, no analytics, and no tracking.

Data stored locally on your device

The following data is stored locally in your browser using Chrome's built-in storage APIs and never leaves your device:

  • Gemini API key — Your API key for Google's Gemini service, stored in Chrome's encrypted sync storage.
  • About me text — Optional background info you provide about yourself, stored in Chrome sync storage.
  • Resume text — If you upload a resume, the extracted text is stored locally in Chrome storage. The original file is not retained.
  • Tone and length preferences — Your default settings.

Third-party services

EasyReach makes two types of external requests, both initiated from your browser:

  • Google Gemini API — When you generate a draft, your prompt and recipient context are sent to Google's Gemini API using your own API key. This is governed by Google's API Terms of Service.
  • LinkedIn — The extension reads publicly visible profile information from LinkedIn pages you visit to provide context for drafting messages. This uses your existing LinkedIn session.

Permissions

  • activeTab / linkedin.com — To read recipient profile information and insert drafted messages.
  • storage — To save your preferences and API key locally.

Data sharing

We do not sell, share, or transfer any user data to third parties. Period.

Changes

If this policy changes, we'll update the date above. Continued use of the extension constitutes acceptance of the updated policy.
